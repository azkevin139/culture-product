# About 📚

Repository containing the Culture Markets Product program, used for creating and managing products within the Culture Markets stack.

This program stores product metadata such as commission rates and escrow account details used during market settlement.

Currently being prepared for sovereign Culture Markets deployments.
> This repository is a Culture Markets fork derived from the Monaco Protocol `protocol-product` codebase and maintained under the Apache License 2.0 with required attribution preserved.
